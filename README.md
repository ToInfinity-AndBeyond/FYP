# Atrial-Fibrillation-ML FYP (Master Thesis)

This workspace contains the Atrial-Fibrillation-ML project, part of a final year project (FYP) and master thesis.

The primary code lives in `Atrial-Fibrillation-ML-main/` and implements a pipeline for:
- PPG-based atrial fibrillation detection using MIMIC PERform AF dataset
- ECG/PPG dataset inspection and synchronization (Zenodo long-term AF data)
- Training (PPG-only and multimodal distillation models)

⚠️ Work in progress: the project is not complete yet and is actively under development.

## Getting started

1. `cd Atrial-Fibrillation-ML-main`
2. `python3 -m venv .venv`
3. `.venv/bin/python -m pip install -r requirements.txt`
